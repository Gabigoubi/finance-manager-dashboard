# FMD - Finance Manager Dashboard

## Objetivo do Projeto

O FMD é um dashboard de Business Intelligence focado em resolver o problema de ineficiência e falta de clareza nos gastos. O objetivo é transformar dados brutos de despesas em insights visuais e acionáveis, permitindo uma tomada de decisão rápida e baseada em fatos.

Este projeto está sendo construído para ser uma ferramenta de portfólio robusta, demonstrando habilidades em arquitetura de software front-end, manipulação de dados e criação de interfaces seguras e eficientes.

---

## Tecnologias Utilizadas

* **Linguagens:** HTML5, CSS3, JavaScript (ES6+).
* **Ferramentas & Ecossistema:**
    * **Vite:** Servidor de desenvolvimento moderno para um fluxo de trabalho rápido.
    * **NPM:** Gerenciamento de pacotes e dependências.
    * **UUID:** Geração de identificadores únicos e seguros para cada despesa.
    * **Git & GitHub:** Versionamento de código e publicação.

---

## Objetivo de Aprendizado

O principal objetivo deste projeto é a consolidação e aplicação profunda dos fundamentos do JavaScript puro (Vanilla JS). As metas de aprendizado incluem:

* Manipulação avançada do DOM (criação e remoção de elementos de forma segura).
* Gerenciamento de estado em uma aplicação front-end.
* Implementação de um sistema de validação de formulários escalável e reutilizável.
* Aplicação de padrões de projeto, como a Delegação de Eventos.
* Prática de código limpo, seguro (prevenção de XSS) e profissional, seguindo as convenções do mercado.
